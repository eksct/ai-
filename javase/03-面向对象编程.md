# 面向对象编程（OOP）

## 三大特性

| 特性 | 说明 | 目的 |
|------|------|------|
| **封装** | 隐藏实现细节，暴露安全接口 | 保护数据，降低耦合 |
| **继承** | 子类复用父类属性和方法 | 代码复用，层次化 |
| **多态** | 同一行为有多种不同实现 | 灵活扩展 |

---

## 类与对象

```java
public class Student {
    // 成员变量（属性）
    private String name;
    private int age;

    // 构造方法
    public Student() {}  // 无参构造
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 方法
    public void study() {
        System.out.println(name + "正在学习");
    }

    // Getter / Setter
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
}
```

## 构造方法

- 与类名相同，无返回值
- 不写则 JVM 提供默认无参构造
- 写了有参构造后，不再提供默认无参构造
- 可以重载（多个构造方法）

```java
public class Person {
    private String name;
    private int age;

    public Person() {}

    public Person(String name) {
        this(name, 0);  // 调用另一个构造方法
    }

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

## 访问修饰符

| 修饰符 | 本类 | 同包 | 子类 | 其他 |
|--------|------|------|------|------|
| `private` | ✅ | ❌ | ❌ | ❌ |
| default（不写） | ✅ | ✅ | ❌ | ❌ |
| `protected` | ✅ | ✅ | ✅ | ❌ |
| `public` | ✅ | ✅ | ✅ | ✅ |

## static 关键字

```java
public class Counter {
    public static int count = 0;   // 静态变量（类变量）
    public static final double PI = 3.14;  // 常量

    private int id;                // 实例变量

    public Counter() {
        this.id = ++count;
    }

    public static void showCount() {  // 静态方法
        System.out.println("实例数: " + count);
        // 静态方法不能直接访问非静态成员
    }

    static {  // 静态代码块（类加载时执行一次）
        System.out.println("类初始化");
    }
}

// 使用
Counter c1 = new Counter();
Counter c2 = new Counter();
System.out.println(Counter.count);  // 2
Counter.showCount();
```

## 继承

```java
public class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }

    public void eat() {
        System.out.println(name + "在吃东西");
    }
}

public class Dog extends Animal {
    public Dog(String name) {
        super(name);  // 调用父类构造方法，必须放在第一行
    }

    @Override
    public void eat() {
        super.eat();  // 调用父类方法
        System.out.println(name + "在啃骨头");
    }

    public void bark() {
        System.out.println(name + "汪汪叫");
    }
}
```

**继承要点：**
- Java 单继承（一个类只能继承一个父类）
- `Object` 是所有类的根父类
- `final` 类不能被继承
- 子类不能继承父类的 private 成员
- 子类可以通过 `super` 访问父类成员

## 多态

```java
// 父类引用指向子类对象
Animal animal = new Dog("小黑");

animal.eat();      // 调用 Dog 的 eat（运行时多态）
// animal.bark();  // 编译错误，父类没有 bark 方法

// 类型检查
if (animal instanceof Dog) {
    Dog dog = (Dog) animal;  // 向下转型
    dog.bark();
}
```

**多态发生条件：**
1. 有继承关系
2. 子类重写父类方法
3. 父类引用指向子类对象

## 抽象类与抽象方法

```java
public abstract class Shape {
    protected String color;

    public Shape(String color) {
        this.color = color;
    }

    // 抽象方法（子类必须实现）
    public abstract double getArea();

    // 普通方法
    public void showColor() {
        System.out.println("颜色: " + color);
    }
}

public class Circle extends Shape {
    private double radius;

    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
}
```

## 接口

```java
public interface Flyable {
    // 常量（public static final）
    int MAX_SPEED = 1000;

    // 抽象方法（public abstract）
    void fly();

    // 默认方法（Java 8+）
    default void takeOff() {
        System.out.println("起飞准备");
    }

    // 静态方法（Java 8+）
    static void showInfo() {
        System.out.println("Flyable 接口");
    }
}

public interface Swimmable {
    void swim();
}

// 多实现
public class Bird implements Flyable, Swimmable {
    @Override
    public void fly() {
        System.out.println("鸟在飞");
    }

    @Override
    public void swim() {
        System.out.println("鸟在游泳");
    }
}
```

### 抽象类 vs 接口

| 对比 | 抽象类 | 接口 |
|------|--------|------|
| 关键字 | `abstract class` | `interface` |
| 继承 | 单继承 | 多实现 |
| 构造方法 | 有 | 无 |
| 成员变量 | 任意 | `public static final` |
| 方法 | 抽象 + 具体 | 抽象 + default + static |
| 设计思想 | is-a | can-do |

## final 关键字

```java
final class MathUtils { /* 不能被继承 */ }

class Parent {
    final void show() { /* 子类不能重写 */ }
}

final int MAX = 100;  // 常量，不能修改
```

## 内部类

```java
public class Outer {
    private int x = 10;

    // 成员内部类
    class Inner {
        void show() {
            System.out.println(x);  // 访问外部类成员
        }
    }

    // 静态内部类
    static class StaticInner {}

    // 局部内部类（方法内）
    void method() {
        class LocalInner {}
    }

    // 匿名内部类（常用于事件监听、线程）
    void useAnonymous() {
        new Thread(new Runnable() {
            @Override
            public void run() {
                System.out.println("匿名内部类");
            }
        }).start();
    }
}

// 使用
Outer outer = new Outer();
Outer.Inner inner = outer.new Inner();
```

## 枚举

```java
public enum Color {
    RED("红色"), GREEN("绿色"), BLUE("蓝色");

    private String chineseName;

    Color(String chineseName) {
        this.chineseName = chineseName;
    }

    public String getChineseName() {
        return chineseName;
    }
}

// 使用
Color c = Color.RED;
System.out.println(c.getChineseName());  // 红色
System.out.println(c.ordinal());         // 0
System.out.println(c.name());            // RED
Color[] colors = Color.values();         // 所有枚举常量
```

## 记录类（Record，Java 14+）

```java
// 自动生成构造方法、getter、equals、hashCode、toString
public record Point(int x, int y) {}

// 使用
Point p = new Point(3, 4);
System.out.println(p.x());   // getter，不是 getX()
System.out.println(p);       // Point[x=3, y=4]
```

## 密封类（Sealed Class，Java 17+）

```java
public sealed class Shape permits Circle, Rectangle, Triangle {
    // 只允许指定的子类继承
}

final class Circle extends Shape {}
final class Rectangle extends Shape {}
sealed class Triangle extends Shape permits RightTriangle {}
final class RightTriangle extends Triangle {}
```
