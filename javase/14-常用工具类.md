### equals 与 hashCode 约定

```java
public class User {
    private Long id;
    private String name;

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        User user = (User) o;
        return Objects.equals(id, user.id) && Objects.equals(name, user.name);
    }

    @Override
    public int hashCode() {
        return Objects.hash(id, name);
    }
}
```

**约定：**
- 两个对象 equals 相等 → hashCode 必须相等
- 两个对象 hashCode 不等 → equals 一定不等
- 两个对象 hashCode 相等 → equals 不一定相等（哈希冲突）

## Objects（工具类）

```java
Objects.equals(a, b);                    // 安全比较（a/b 可为 null）
Objects.hash(a, b, c);                   // 生成哈希码
Objects.requireNonNull(obj);             // 非空检查
Objects.requireNonNull(obj, "msg");      // 带消息的非空检查
Objects.isNull(obj);
Objects.nonNull(obj);
Objects.toString(obj, "default");        // null 安全 toString
```

## Arrays

```java
int[] arr = {3, 1, 4, 1, 5, 9};

Arrays.sort(arr);                        // 排序
Arrays.binarySearch(arr, 4);             // 二分查找（需排序）
Arrays.copyOf(arr, 10);                  // 扩容/复制
Arrays.copyOfRange(arr, 1, 4);           // 范围复制
Arrays.fill(arr, 0);                     // 填充
Arrays.toString(arr);                    // 转字符串
Arrays.deepToString(matrix);             // 多维数组转字符串
Arrays.equals(arr1, arr2);              // 比较
Arrays.deepEquals(matrix1, matrix2);     // 多维数组比较
Arrays.asList("a", "b", "c");            // 转 List（固定大小）
Arrays.parallelSort(arr);                // 并行排序（大数组）
Arrays.setAll(arr, i -> i * 2);          // 设置所有元素
```

## Collections

```java
List<String> list = new ArrayList<>();

Collections.sort(list);                   // 排序
Collections.reverse(list);                // 反转
Collections.shuffle(list);                // 打乱
Collections.rotate(list, 2);              // 旋转

Collections.binarySearch(list, "key");    // 二分查找
Collections.max(list);
Collections.min(list);
Collections.frequency(list, "a");         // 出现次数
Collections.disjoint(list1, list2);       // 是否有交集

Collections.fill(list, "default");        // 填充
Collections.copy(dest, src);              // 复制

// 包装为不可变
Collections.unmodifiableList(list);
Collections.unmodifiableSet(set);
Collections.unmodifiableMap(map);

// 线程安全包装
Collections.synchronizedList(list);
Collections.synchronizedMap(map);

// 单元素集合
Collections.singletonList("a");
Collections.singleton("a");
Collections.singletonMap("key", "value");

// 空集合
Collections.emptyList();
Collections.emptySet();
Collections.emptyMap();
```

## Random

```java
// 常用
Random rand = new Random();
rand.nextInt();                    // 随机整数
rand.nextInt(100);                 // [0, 100)
rand.nextLong();
rand.nextDouble();                 // [0.0, 1.0)
rand.nextBoolean();
rand.ints(10, 0, 100).toArray();  // 10 个 [0,100) 的整数

// ThreadLocalRandom（多线程场景性能更好）
ThreadLocalRandom.current().nextInt(100);

// SecureRandom（密码学安全）
SecureRandom sr = new SecureRandom();
byte[] key = new byte[32];
sr.nextBytes(key);
```

## Properties

```java
// 读取配置文件
Properties props = new Properties();
try (FileInputStream fis = new FileInputStream("config.properties")) {
    props.load(fis);
}

String url = props.getProperty("db.url");
String port = props.getProperty("db.port", "3306");  // 默认值

// 写入
props.setProperty("app.name", "MyApp");
try (FileOutputStream fos = new FileOutputStream("config.properties")) {
    props.store(fos, "Updated");
}
```

**config.properties 格式：**
```properties
db.url=jdbc:mysql://localhost:3306/mydb
db.user=root
db.password=secret
```
