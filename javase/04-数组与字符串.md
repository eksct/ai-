# 数组与字符串

## 数组

### 声明与初始化

```java
// 声明
int[] arr1;
int arr2[];

// 静态初始化
int[] arr = {1, 2, 3, 4, 5};
String[] names = {"Alice", "Bob"};

// 动态初始化
int[] arr = new int[5];           // 默认值 0
int[] arr2 = new int[]{1, 2, 3};  // 指定初始值
```

### 常用操作

```java
int[] arr = {3, 1, 4, 1, 5, 9};

// 遍历
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
for (int num : arr) {
    System.out.println(num);
}

// 复制
int[] copy = Arrays.copyOf(arr, arr.length);
int[] copyRange = Arrays.copyOfRange(arr, 1, 4);

// 排序
Arrays.sort(arr);

// 查找（需先排序）
int index = Arrays.binarySearch(arr, 4);

// 填充
Arrays.fill(arr, 0);

// 转字符串
System.out.println(Arrays.toString(arr));

// 比较
boolean eq = Arrays.equals(arr, copy);

// 二维数组
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};
System.out.println(matrix[0][1]);  // 2
```

## 字符串

### String（不可变）

```java
// 创建
String s1 = "Hello";                    // 字符串常量池
String s2 = new String("Hello");        // 堆内存
char[] arr = {'H', 'e', 'l', 'l', 'o'};
String s3 = new String(arr);

// 常用方法
s.length()                         // 长度
s.charAt(0)                        // 获取字符
s.toCharArray()                    // 转字符数组
s.equals("Hello")                  // 比较内容（重要！）
s.equalsIgnoreCase("hello")        // 忽略大小写比较
s.contains("ell")                  // 是否包含子串
s.startsWith("He")                 // 前缀
s.endsWith("lo")                   // 后缀
s.indexOf('e')                     // 查找位置
s.lastIndexOf('l')
s.substring(1, 4)                  // 截取 [1,4)
s.replace('l', 'p')                // 替换
s.replaceAll("\\d", "*")           // 正则替换
s.split(",")                       // 分割
s.trim()                           // 去空格
s.toUpperCase() / toLowerCase()
s.isEmpty()                        // 是否为空
s.isBlank()                        // 是否为空白（Java 11+）
s.strip()                          // 去空格（支持 Unicode，Java 11+）
"  Hello  ".strip()                // "Hello"
```

### StringBuilder（可变，线程不安全）

```java
StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" ");
sb.append("World");
sb.insert(5, ",");
sb.delete(5, 6);
sb.reverse();
sb.replace(0, 5, "Hi");
String result = sb.toString();
```

### StringBuffer（可变，线程安全）
- 用法与 `StringBuilder` 相同
- 方法都加了 `synchronized`，性能略低

### String vs StringBuilder vs StringBuffer

| 类 | 可变性 | 线程安全 | 性能 |
|------|--------|----------|------|
| `String` | 不可变 | 安全 | 频繁修改时差 |
| `StringBuilder` | 可变 | 不安全 | 最快 |
| `StringBuffer` | 可变 | 安全 | 较慢 |

```java
// 频繁拼接字符串的正确做法
// ❌ 不推荐
String s = "";
for (int i = 0; i < 1000; i++) {
    s += i;  // 每次创建新 String 对象
}

// ✅ 推荐
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i);
}
String s = sb.toString();
```

## 字符串常量池

```java
String s1 = "Hello";
String s2 = "Hello";
String s3 = new String("Hello");

System.out.println(s1 == s2);      // true（常量池同一对象）
System.out.println(s1 == s3);      // false（堆中新对象）
System.out.println(s1.equals(s3)); // true（内容相同）

// intern()：将字符串放入常量池
String s4 = s3.intern();
System.out.println(s1 == s4);      // true
```

## 格式化字符串

```java
// String.format
String s = String.format("名称: %s, 数量: %d, 价格: %.2f", name, count, price);

// printf
System.out.printf("名称: %s, 数量: %d%n", name, count);

// formatted (Java 15+)
String s = "名称: %s, 数量: %d".formatted(name, count);
```

### 格式化符号

| 符号 | 说明 |
|------|------|
| `%s` | 字符串 |
| `%d` | 整数 |
| `%f` | 浮点数 |
| `%.2f` | 两位小数 |
| `%x` | 十六进制 |
| `%tY` | 年份 |
| `%tm` | 月份 |
| `%td` | 日期 |
| `%n` | 换行 |

## 正则表达式

```java
// 匹配
"abc123".matches("\\w+\\d+");      // true

// 分割
"a,b,c".split(",");                 // ["a", "b", "c"]

// 替换
"abc123def".replaceAll("\\d+", "#"); // "abc#def"

// Pattern/Matcher
Pattern pattern = Pattern.compile("(\\w+)@(\\w+)");
Matcher matcher = pattern.matcher("user@example.com");
if (matcher.matches()) {
    System.out.println(matcher.group(1));  // user
    System.out.println(matcher.group(2));  // example
}
```
