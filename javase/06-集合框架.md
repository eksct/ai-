# 集合框架

## 集合体系

```
Collection（接口）
 ├── List（有序可重复）
 │    ├── ArrayList（数组实现）
 │    ├── LinkedList（链表实现）
 │    └── Vector（线程安全，已不推荐）
 │         └── Stack
 ├── Set（无序不可重复）
 │    ├── HashSet（哈希表）
 │    │    └── LinkedHashSet（维护插入顺序）
 │    └── TreeSet（红黑树，排序）
 └── Queue（队列）
      ├── LinkedList（双端队列）
      ├── PriorityQueue（优先队列）
      └── ArrayDeque（数组双端队列）

Map（接口）
 ├── HashMap（哈希表）
 │    └── LinkedHashMap（维护插入/访问顺序）
 ├── TreeMap（红黑树，排序）
 └── Hashtable（线程安全，已不推荐）
```

## List

### ArrayList

```java
// 创建
List<String> list1 = new ArrayList<>();
List<String> list2 = new ArrayList<>(100);  // 指定初始容量
List<String> list3 = Arrays.asList("a", "b", "c");  // 固定大小
List<String> list4 = List.of("a", "b", "c");         // 不可变（Java 9+）

// 增删改查
list.add("a");
list.add(1, "b");     // 指定位置插入
list.addAll(otherList);
list.get(0);
list.set(0, "new");
list.remove(0);        // 按索引删除
list.remove("a");      // 按对象删除
list.clear();

// 查询
list.size();
list.isEmpty();
list.contains("a");
list.indexOf("a");
list.lastIndexOf("a");

// 遍历
for (String s : list) { }
for (int i = 0; i < list.size(); i++) { }
Iterator<String> it = list.iterator();
list.forEach(System.out::println);  // Java 8+

// 排序
list.sort(Comparator.naturalOrder());
list.sort((a, b) -> a.compareTo(b));
```

### ArrayList vs LinkedList

| 对比 | ArrayList | LinkedList |
|------|-----------|------------|
| 底层结构 | 动态数组 | 双向链表 |
| 随机访问（get） | O(1) ✅ | O(n) ❌ |
| 插入/删除（尾部） | O(1) ✅ | O(1) ✅ |
| 插入/删除（中间） | O(n) ❌ | O(1) ✅ |
| 内存占用 | 少 | 多（存储前后指针） |

## Set

```java
// HashSet（无序）
Set<String> set = new HashSet<>();
set.add("apple");
set.add("banana");
set.add("apple");           // 重复元素不会添加
set.contains("apple");      // true
set.remove("apple");

// LinkedHashSet（维护插入顺序）
Set<String> linkedSet = new LinkedHashSet<>();

// TreeSet（自然排序/Comparator）
Set<String> treeSet = new TreeSet<>();
Set<String> treeSet2 = new TreeSet<>(Comparator.reverseOrder());
treeSet.first();            // 最小
treeSet.last();             // 最大
treeSet.headSet("c");       // 小于 c 的子集
treeSet.tailSet("b");       // 大于等于 b 的子集
```

### HashSet 原理
- 底层基于 `HashMap`
- 通过 `hashCode()` 确定存储位置
- 通过 `equals()` 判断元素是否相同
- 自定义类存入 HashSet 必须重写 `hashCode()` 和 `equals()`

## Queue / Deque

```java
// Queue
Queue<String> queue = new LinkedList<>();
queue.offer("a");           // 入队（推荐）
queue.add("b");             // 入队（满时抛异常）
queue.poll();               // 出队，队空返回 null
queue.remove();             // 出队，队空抛异常
queue.peek();               // 查看队首，空返回 null
queue.element();            // 查看队首，空抛异常

// Deque（双端队列）
Deque<String> deque = new ArrayDeque<>();
deque.addFirst("a");
deque.addLast("b");
deque.removeFirst();
deque.removeLast();
deque.peekFirst();
deque.peekLast();

// 作为栈使用
Deque<String> stack = new ArrayDeque<>();
stack.push("a");            // 入栈
stack.pop();                // 出栈
stack.peek();               // 查看栈顶
```

## Map

```java
// HashMap
Map<String, Integer> map = new HashMap<>();
map.put("apple", 10);
map.put("banana", 20);
map.putIfAbsent("apple", 5);       // 不存在才放入
map.get("apple");                   // 10
map.getOrDefault("orange", 0);     // 不存在返回默认值
map.containsKey("apple");
map.containsValue(10);
map.remove("apple");
map.replace("apple", 10, 15);      // 值匹配才替换

// 遍历
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    entry.getKey();
    entry.getValue();
}
for (String key : map.keySet()) { }
for (Integer value : map.values()) { }
map.forEach((k, v) -> System.out.println(k + "=" + v));

// LinkedHashMap（维护插入/访问顺序）
Map<String, Integer> linkedMap = new LinkedHashMap<>(16, 0.75f, true);  // access-order

// TreeMap（排序）
TreeMap<String, Integer> treeMap = new TreeMap<>();
treeMap.firstKey();          // 最小键
treeMap.lastKey();           // 最大键
treeMap.lowerKey("c");       // 小于 c 的最大键
treeMap.higherKey("b");      // 大于 b 的最小键
treeMap.subMap("a", "d");    // 子集 [a, d)
```

### HashMap 原理

| 版本 | 底层结构 |
|------|----------|
| Java 7 | 数组 + 链表 |
| Java 8+ | 数组 + 链表 / 红黑树（链表长度 ≥8 转红黑树） |

- 默认容量 16，负载因子 0.75
- 当元素数 > 容量 × 负载因子时扩容（2 倍）
- 允许 key/value 为 null
- 线程不安全（并发用 `ConcurrentHashMap`）

## 集合工具类 Collections

```java
List<String> list = new ArrayList<>();

// 排序
Collections.sort(list);
Collections.sort(list, Comparator.reverseOrder());
Collections.reverse(list);

// 查找
Collections.binarySearch(list, "key");  // 需先排序
Collections.max(list);
Collections.min(list);

// 填充/复制
Collections.fill(list, "default");
Collections.copy(dest, src);

// 不可变集合
List<String> unmodifiable = Collections.unmodifiableList(list);
// 任何修改操作会抛 UnsupportedOperationException

// 线程安全集合
List<String> syncList = Collections.synchronizedList(list);
```

## 不可变集合（Java 9+）

```java
List<String> list = List.of("a", "b", "c");
Set<String> set = Set.of("a", "b", "c");

Map<String, Integer> map = Map.ofEntries(
    Map.entry("a", 1),
    Map.entry("b", 2)
);
```

## 集合选型速查

```
需要什么？
 ├── 键值对？        → Map
 │    ├── 有序？     → TreeMap / LinkedHashMap
 │    ├── 线程安全？ → ConcurrentHashMap
 │    └── 否则      → HashMap
 ├── 元素唯一？      → Set
 │    ├── 有序？     → TreeSet / LinkedHashSet
 │    └── 否则      → HashSet
 ├── 频繁随机访问？  → ArrayList
 ├── 频繁插入删除？  → LinkedList
 ├── FIFO 队列？     → LinkedList / ArrayDeque
 └── LIFO 栈？       → ArrayDeque
```
