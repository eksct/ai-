# I/O 流

## 字节流 vs 字符流

```
字节流（处理二进制文件：图片、视频、zip）
  InputStream
    ├── FileInputStream
    ├── BufferedInputStream
    ├── ByteArrayInputStream
    ├── ObjectInputStream
    └── DataInputStream

  OutputStream
    ├── FileOutputStream
    ├── BufferedOutputStream
    ├── ByteArrayOutputStream
    ├── ObjectOutputStream
    └── DataOutputStream

字符流（处理文本文件：txt, csv, json）
  Reader
    ├── FileReader
    ├── BufferedReader
    ├── InputStreamReader
    ├── StringReader
    └── CharArrayReader

  Writer
    ├── FileWriter
    ├── BufferedWriter
    ├── OutputStreamWriter
    ├── StringWriter
    └── CharArrayWriter
```

## 文件操作

### File（文件和目录路径名）

```java
File file = new File("test.txt");
File dir = new File("docs/subdir");

// 路径信息
file.getName();
file.getPath();
file.getAbsolutePath();
file.getCanonicalPath();   // 标准化路径

// 文件信息
file.exists();
file.isFile();
file.isDirectory();
file.length();             // 文件大小
file.lastModified();

// 操作
file.createNewFile();      // 创建新文件
file.delete();             // 删除
file.renameTo(newName);
dir.mkdir();               // 创建单级目录
dir.mkdirs();              // 创建多级目录

// 列出目录
String[] names = dir.list();
File[] files = dir.listFiles();
File[] txts = dir.listFiles((f, name) -> name.endsWith(".txt"));
```

### Files / Paths（NIO，Java 7+）

```java
Path path = Paths.get("docs", "subdir", "test.txt");
Path path = Paths.get("docs/subdir/test.txt");

// 读写
byte[] bytes = Files.readAllBytes(path);        // 读二进制
List<String> lines = Files.readAllLines(path);  // 读文本
Files.write(path, "hello".getBytes());           // 写二进制
Files.write(path, lines);                        // 写文本
Files.writeString(path, "hello");               // 写字符串（Java 11+）

// 操作
Files.createFile(path);
Files.createDirectories(path.getParent());
Files.copy(source, target, StandardCopyOption.REPLACE_EXISTING);
Files.move(source, target);
Files.delete(path);
Files.exists(path);
Files.isDirectory(path);
Files.size(path);

// 遍历目录
Files.walk(Paths.get("docs"))
     .filter(p -> p.toString().endsWith(".txt"))
     .forEach(System.out::println);
```

## 文本文件读写

### 传统方式

```java
// 读（try-with-resources）
try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }
}

// 写
try (BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"))) {
    writer.write("第一行");
    writer.newLine();
    writer.write("第二行");
}

// 追加模式
try (FileWriter fw = new FileWriter("log.txt", true);
     BufferedWriter bw = new BufferedWriter(fw)) {
    bw.write("新的日志行");
    bw.newLine();
}
```

### 指定编码（重要）

```java
// ❌ 不推荐（使用平台默认编码，可能乱码）
new BufferedReader(new FileReader("test.txt"));

// ✅ 推荐（指定 UTF-8）
try (BufferedReader reader = new BufferedReader(
        new InputStreamReader(new FileInputStream("test.txt"), StandardCharsets.UTF_8))) {
    // ...
}

// Java 11+ 更方便
Files.readString(Paths.get("test.txt"));  // 默认 UTF-8
Files.writeString(Paths.get("test.txt"), "内容");
```

## 二进制文件读写

```java
// 读
try (FileInputStream fis = new FileInputStream("image.jpg");
     BufferedInputStream bis = new BufferedInputStream(fis)) {
    byte[] buffer = new byte[8192];
    int bytesRead;
    while ((bytesRead = bis.read(buffer)) != -1) {
        // 处理 buffer 中前 bytesRead 个字节
    }
}

// 写
try (FileOutputStream fos = new FileOutputStream("output.jpg");
     BufferedOutputStream bos = new BufferedOutputStream(fos)) {
    bos.write(data);
    bos.flush();  // 刷新缓冲区
}

// NIO 方式（更简洁）
byte[] data = Files.readAllBytes(Paths.get("source.jpg"));
Files.write(Paths.get("dest.jpg"), data);
```

## 对象序列化

```java
// 需要实现 Serializable 接口
public class User implements Serializable {
    private static final long serialVersionUID = 1L;  // 版本号

    private String name;
    private transient String password;  // transient 字段不序列化
    private int age;
    // getter/setter...
}

// 序列化
try (ObjectOutputStream oos = new ObjectOutputStream(
        new FileOutputStream("user.dat"))) {
    oos.writeObject(new User("Alice", "secret", 25));
}

// 反序列化
try (ObjectInputStream ois = new ObjectInputStream(
        new FileInputStream("user.dat"))) {
    User user = (User) ois.readObject();
}

// 自定义序列化（writeObject / readObject）
private void writeObject(ObjectOutputStream oos) throws IOException {
    oos.defaultWriteObject();     // 默认序列化
    oos.writeObject(encrypt(password));  // 加密后写入
}

private void readObject(ObjectInputStream ois) throws IOException, ClassNotFoundException {
    ois.defaultReadObject();      // 默认反序列化
    this.password = decrypt((String) ois.readObject());  // 解密
}
```

## 装饰器模式

Java I/O 的设计模式是**装饰器模式**，通过多层包装增强功能：

```java
// 从文件读取 → 缓冲 → 读取行
BufferedReader reader = new BufferedReader(
    new InputStreamReader(
        new FileInputStream("test.txt"), StandardCharsets.UTF_8));

// 层层包装：
// FileInputStream      → 从文件读字节
// InputStreamReader   → 字节→字符
// BufferedReader      → 缓冲（读取行、按行读）
```

## NIO（Java NIO）

### Channel + Buffer

```java
// 读文件
try (FileChannel channel = FileChannel.open(Paths.get("test.txt"), StandardOpenOption.READ)) {
    ByteBuffer buffer = ByteBuffer.allocate(1024);
    while (channel.read(buffer) != -1) {
        buffer.flip();  // 切换到读模式
        while (buffer.hasRemaining()) {
            System.out.print((char) buffer.get());
        }
        buffer.clear();  // 切换回写模式
    }
}

// 写文件
try (FileChannel channel = FileChannel.open(
        Paths.get("output.txt"),
        StandardOpenOption.CREATE, StandardOpenOption.WRITE)) {
    ByteBuffer buffer = ByteBuffer.wrap("Hello, NIO!".getBytes());
    channel.write(buffer);
}
```

## I/O 选型指南

| 场景 | 推荐方式 |
|------|----------|
| 读小文本文件 | `Files.readString(path)`（Java 11+） |
| 读大文本文件 | `BufferedReader` + `FileReader` |
| 写小文本文件 | `Files.writeString(path, content)` |
| 读二进制文件 | `Files.readAllBytes(path)` / `BufferedInputStream` |
| 对象序列化 | `ObjectOutputStream` / `ObjectInputStream` |
| 需要编码控制 | `InputStreamReader` / `OutputStreamWriter` |
| 高性能 I/O | FileChannel + ByteBuffer（NIO） |
