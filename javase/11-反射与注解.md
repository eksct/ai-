# 反射与注解

## 反射

反射是在运行时动态获取类的信息并操作类成员的能力。

### 获取 Class 对象

```java
// 方式一：Class.forName（最常用）
Class<?> clazz = Class.forName("java.util.ArrayList");

// 方式二：.class 语法
Class<?> clazz = String.class;

// 方式三：getClass()
String s = "hello";
Class<?> clazz = s.getClass();

// 方式四：通过对象获取
Class<?> clazz = Integer.TYPE;  // 基本类型
```

### 获取类信息

```java
Class<?> clazz = User.class;

// 类名
clazz.getName();              // com.example.User（全限定名）
clazz.getSimpleName();        // User
clazz.getPackage().getName(); // com.example

// 修饰符
clazz.getModifiers();         // 用 Modifier 解析
Modifier.isPublic(clazz.getModifiers());

// 父类/接口
clazz.getSuperclass();        // 父类
clazz.getInterfaces();        // 接口
```

### 获取构造方法

```java
// 获取所有公共构造方法
Constructor<?>[] constructors = clazz.getConstructors();

// 获取指定参数类型的构造方法
Constructor<User> constructor = clazz.getConstructor(String.class, int.class);

// 创建实例
User user = constructor.newInstance("Alice", 25);

// 无参构造
User user2 = clazz.getDeclaredConstructor().newInstance();
```

### 获取成员变量

```java
// 公共字段（含继承的）
Field[] fields = clazz.getFields();

// 本类所有字段（含 private，不含继承的）
Field[] declaredFields = clazz.getDeclaredFields();

// 获取指定字段
Field field = clazz.getDeclaredField("name");
field.setAccessible(true);     // 突破 private

// 读取/设置值
User user = new User("Alice", 25);
String name = (String) field.get(user);
field.set(user, "Bob");
```

### 获取方法

```java
// 公共方法（含继承的）
Method[] methods = clazz.getMethods();

// 本类所有方法（含 private）
Method[] declaredMethods = clazz.getDeclaredMethods();

// 获取指定方法
Method method = clazz.getMethod("getName");
Method method2 = clazz.getDeclaredMethod("setName", String.class);
method2.setAccessible(true);

// 调用方法
User user = new User();
String name = (String) method.invoke(user);
method2.invoke(user, "Alice");

// 调用静态方法
Method staticMethod = clazz.getMethod("staticMethod");
staticMethod.invoke(null);  // 第一个参数传 null
```

### 操作数组

```java
// 创建数组
int[] arr = (int[]) Array.newInstance(int.class, 5);

// 设置/获取值
Array.set(arr, 0, 10);
int val = Array.getInt(arr, 0);
```

### 反射的应用

```java
// 1. 实例化（框架中通过配置创建对象）
String className = config.getProperty("service.class");
Object service = Class.forName(className).getDeclaredConstructor().newInstance();

// 2. 调用方法（通用的处理方法）
public Object invokeMethod(Object obj, String methodName, Object... args) {
    Class<?>[] paramTypes = Arrays.stream(args)
        .map(Object::getClass)
        .toArray(Class<?>[]::new);
    Method method = obj.getClass().getMethod(methodName, paramTypes);
    return method.invoke(obj, args);
}

// 3. 读取注解（见下方）
```

## 注解

### 元注解

```java
@Target(ElementType.TYPE)       // 注解可用的目标位置
@Retention(RetentionPolicy.RUNTIME)  // 保留策略
@Documented                     // 包含在 javadoc 中
@Inherited                      // 子类继承
@Repeatable(Annotations.class)  // 可重复注解
public @interface MyAnnotation {
    String value() default "";
}
```

### @Target 取值

| 值 | 目标 |
|------|------|
| `TYPE` | 类、接口、枚举 |
| `FIELD` | 字段 |
| `METHOD` | 方法 |
| `PARAMETER` | 参数 |
| `CONSTRUCTOR` | 构造方法 |
| `ANNOTATION_TYPE` | 注解 |
| `PACKAGE` | 包 |
| `TYPE_PARAMETER` | 类型参数（Java 8+） |
| `TYPE_USE` | 类型使用（Java 8+） |

### @Retention 取值

| 值 | 说明 |
|------|------|
| `SOURCE` | 仅源码，编译后丢弃 |
| `CLASS` | 编译到字节码，运行时不可反射 |
| `RUNTIME` | 编译到字节码，运行时可通过反射读取 |

### 自定义注解

```java
@Target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
public @interface Loggable {
    String level() default "INFO";
    boolean logParams() default true;
    boolean logResult() default true;
}
```

### 使用自定义注解

```java
@Loggable(level = "DEBUG")
public class UserService {

    @Loggable(logParams = true, logResult = true)
    public User findUser(Long id) {
        // ...
        return user;
    }
}
```

### 读取注解（反射）

```java
// 读取类注解
Class<UserService> clazz = UserService.class;
if (clazz.isAnnotationPresent(Loggable.class)) {
    Loggable loggable = clazz.getAnnotation(Loggable.class);
    System.out.println(loggable.level());
}

// 读取方法注解
Method method = clazz.getMethod("findUser", Long.class);
if (method.isAnnotationPresent(Loggable.class)) {
    Loggable loggable = method.getAnnotation(Loggable.class);
    System.out.println(loggable.level());
    System.out.println(loggable.logParams());
}

// 遍历方法注解
for (Method m : clazz.getMethods()) {
    Loggable log = m.getAnnotation(Loggable.class);
    if (log != null) {
        System.out.println(m.getName() + " → " + log.level());
    }
}
```

### 内置注解

```java
@Override           // 重写父类方法
@Deprecated         // 已废弃
@SuppressWarnings("unchecked")  // 抑制编译警告
@SafeVarargs        // 安全的可变参数
@FunctionalInterface  // 函数式接口
```

## 反射与注解实战：简易 ORM

```java
// 自定义 ORM 注解
@Target(ElementType.TYPE)
@Retention(RetentionPolicy.RUNTIME)
public @interface Table {
    String name();
}

@Target(ElementType.FIELD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Column {
    String name();
    boolean primaryKey() default false;
}

// 实体类
@Table(name = "users")
public class User {
    @Column(name = "id", primaryKey = true)
    private Long id;

    @Column(name = "user_name")
    private String userName;

    @Column(name = "age")
    private int age;
    // getter/setter...
}

// 简易 ORM 工具
public class SimpleORM {
    public static String buildInsertSQL(Object obj) throws Exception {
        Class<?> clazz = obj.getClass();
        Table table = clazz.getAnnotation(Table.class);
        if (table == null) {
            throw new IllegalArgumentException("缺少 @Table 注解");
        }

        StringBuilder sb = new StringBuilder();
        sb.append("INSERT INTO ").append(table.name()).append(" (");

        List<String> columns = new ArrayList<>();
        List<String> values = new ArrayList<>();

        for (Field field : clazz.getDeclaredFields()) {
            Column column = field.getAnnotation(Column.class);
            if (column != null) {
                field.setAccessible(true);
                columns.add(column.name());
                Object value = field.get(obj);
                values.add("'" + value.toString() + "'");
            }
        }

        sb.append(String.join(", ", columns));
        sb.append(") VALUES (");
        sb.append(String.join(", ", values));
        sb.append(")");

        return sb.toString();
    }
}

// 使用
User user = new User(1L, "Alice", 25);
String sql = SimpleORM.buildInsertSQL(user);
// INSERT INTO users (id, user_name, age) VALUES ('1', 'Alice', '25')
```
