# 泛型

## 为什么需要泛型

```java
// 没有泛型时（强制转型 + 类型不安全）
List list = new ArrayList();
list.add("hello");
list.add(123);              // 不小心混入其他类型
String s = (String) list.get(0);  // 需要强制转型
String s2 = (String) list.get(1); // ClassCastException!

// 使用泛型（类型安全 + 无需转型）
List<String> list = new ArrayList<>();
list.add("hello");
// list.add(123);            // 编译错误
String s = list.get(0);      // 无需强制转型
```

## 泛型类

```java
public class Box<T> {
    private T item;

    public void set(T item) {
        this.item = item;
    }

    public T get() {
        return item;
    }
}

// 多个类型参数
public class Pair<K, V> {
    private K key;
    private V value;

    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }
    // getter/setter...
}

// 使用
Box<String> stringBox = new Box<>();
stringBox.set("hello");

Pair<String, Integer> pair = new Pair<>("age", 25);  // 菱形运算符 <>
```

## 泛型接口

```java
public interface Repository<T, ID> {
    T findById(ID id);
    List<T> findAll();
    void save(T entity);
    void delete(ID id);
}

// 实现泛型接口
public class UserRepository implements Repository<User, Long> {
    @Override
    public User findById(Long id) { ... }

    @Override
    public List<User> findAll() { ... }

    @Override
    public void save(User user) { ... }

    @Override
    public void delete(Long id) { ... }
}
```

## 泛型方法

```java
public class Utils {
    // 泛型方法，类型参数在返回值前声明
    public static <T> T getMiddle(T[] array) {
        return array[array.length / 2];
    }

    public static <T extends Comparable<T>> T max(T a, T b) {
        return a.compareTo(b) > 0 ? a : b;
    }

    // 多个类型参数
    public static <K, V> void printEntry(K key, V value) {
        System.out.println(key + " = " + value);
    }
}

// 使用
String middle = Utils.getMiddle(new String[]{"a", "b", "c"});
int max = Utils.max(3, 5);  // 类型推断
```

## 类型通配符

### 无界通配符 `?`

```java
// 可以处理任何类型的集合
public static void printList(List<?> list) {
    for (Object elem : list) {
        System.out.println(elem);
    }
}

// 只能读取，不能写入（除了 null）
List<?> list = new ArrayList<String>();
Object obj = list.get(0);   // ✅ 可以读
// list.add("hello");       // ❌ 不可以写
list.add(null);             // ✅ 只能加 null
```

### 上界通配符 `? extends T`

```java
// 读取 T 或 T 的子类型
public static double sumOfList(List<? extends Number> list) {
    double sum = 0;
    for (Number n : list) {
        sum += n.doubleValue();
    }
    return sum;
}

// 只能读，不能写
List<? extends Number> nums = new ArrayList<Integer>();
Number n = nums.get(0);     // ✅ 可以读
// nums.add(1);             // ❌ 不可以写（编译器不知道具体子类型）
```

### 下界通配符 `? super T`

```java
// 写入 T 或 T 的父类型
public static void addNumbers(List<? super Integer> list) {
    list.add(1);   // ✅ 可以写 Integer
    list.add(2);
    // Integer n = list.get(0);  // ❌ 不可以读为 Integer
    Object obj = list.get(0);     // ✅ 只能读为 Object
}
```

### PECS 原则（Producer Extends, Consumer Super）

```
? extends T  → Producer（生产者），只读不写
? super T    → Consumer（消费者），只写不读
```

```java
// 示例：Collections.copy
public static <T> void copy(List<? super T> dest, List<? extends T> src) {
    // src 是生产者（只读），dest 是消费者（只写）
    for (int i = 0; i < src.size(); i++) {
        dest.set(i, src.get(i));
    }
}
```

## 类型擦除

Java 的泛型是在编译期实现的，运行时被擦除：

```java
// 编译前
List<String> list1 = new ArrayList<>();
List<Integer> list2 = new ArrayList<>();
System.out.println(list1.getClass() == list2.getClass());  // true

// 编译后（类型擦除）
List list1 = new ArrayList();
List list2 = new ArrayList();
```

**擦除规则：**
- `T` → `Object`
- `T extends Comparable` → `Comparable`
- `T extends Number & Comparable` → `Number`

**桥接方法：**
```java
// 编译器在必要时生成桥接方法保持多态
// 编译前
public class MyComparator implements Comparator<Integer> {
    public int compare(Integer a, Integer b) { ... }
}
// 编译后（自动生成桥接方法）
public class MyComparator implements Comparator {
    public int compare(Object a, Object b) {   // 桥接方法
        return compare((Integer) a, (Integer) b);
    }
    public int compare(Integer a, Integer b) { ... }
}
```

## 泛型限制

```java
// 1. 不能实例化类型参数
T obj = new T();        // ❌

// 2. 不能创建泛型数组
T[] arr = new T[10];    // ❌
T[] arr = (T[]) new Object[10];  // ✅ 但类型不安全

// 3. 静态成员不能使用类的类型参数
public class Box<T> {
    private static T item;  // ❌ 静态成员不能使用 T
    public static T get() { // ❌ 静态方法不能使用类的 T
        return item;
    }
}

// 4. 不能用于基本类型
List<int> list;   // ❌ 必须用包装类
List<Integer> list;  // ✅

// 5. 不能用在异常 catch
try { } catch (T e) { }  // ❌

// 6. instanceof 不能用于泛型类型
if (list instanceof List<String>) { }  // ❌ 类型已擦除
```

## 泛型与反射

```java
// 获取泛型类型（运行时通过 TypeToken 等方式）
public class TypeToken<T> {
    private final Type type;

    protected TypeToken() {
        Type superclass = getClass().getGenericSuperclass();
        this.type = ((ParameterizedType) superclass).getActualTypeArguments()[0];
    }

    public Type getType() { return type; }
}

// 使用
TypeToken<List<String>> token = new TypeToken<List<String>>() {};
System.out.println(token.getType());  // java.util.List<java.lang.String>
```
