# 饮食营养常识

## 每日膳食结构

### 中国居民平衡膳食宝塔（2022）

```
▲ 油盐类：油 25-30g，盐 <5g/天
▲ 奶及奶制品：300-500g/天
  大豆及坚果：25-35g/天
▲ 动物性食物：120-200g/天
  （鱼虾 40-75g，畜禽肉 40-75g，蛋类 1个/天）
▲ 蔬菜水果：蔬菜 300-500g，水果 200-350g/天
▲ 谷薯类：250-400g/天
  （全谷物和杂豆 50-150g，薯类 50-100g）
▼ 饮水：1500-1700ml/天
```

## 营养素基础

| 营养素 | 功能 | 食物来源 |
|--------|------|----------|
| 碳水化合物 | 主要能量来源 | 米饭、面条、全麦面包、薯类 |
| 蛋白质 | 修复组织、免疫 | 鸡蛋、牛奶、鱼肉、豆制品 |
| 脂肪 | 能量储备、激素合成 | 植物油、坚果、鱼油 |
| 膳食纤维 | 肠道健康 | 蔬菜、水果、全谷物 |
| 维生素 C | 抗氧化、免疫 | 柑橘、猕猴桃、青椒 |
| 维生素 D | 钙吸收、骨骼 | 晒太阳、蛋黄、鱼肝油 |
| 钙 | 骨骼牙齿 | 牛奶、豆制品、深色蔬菜 |
| 铁 | 血红蛋白 | 红肉、猪肝、菠菜 |
| 锌 | 免疫、伤口愈合 | 生蚝、瘦肉、坚果 |

## 饮食原则

### 三餐分配
- 早餐 30%：吃好（高蛋白、高纤维）
- 午餐 40%：吃饱（均衡搭配）
- 晚餐 30%：吃少（清淡、易消化）

### 健康饮食原则
1. **食物多样**：每天吃 12 种以上食物，每周 25 种以上
2. **粗细搭配**：全谷物占主食 1/3
3. **彩虹原则**：多种颜色的蔬菜水果搭配
4. **少油少盐**：烹饪油 25-30g/天，盐 <5g/天
5. **足量饮水**：少量多次，不喝含糖饮料
6. **细嚼慢咽**：每口咀嚼 20-30 次
7. **七分饱**：不暴饮暴食

### 不好的饮食习惯
- ❌ 不吃早餐
- ❌ 晚餐过晚过饱
- ❌ 常吃外卖、油炸食品
- ❌ 含糖饮料当水喝
- ❌ 边吃饭边看手机

## 食品安全

### 食物储存
- 生熟分开：生肉、海鲜与熟食分开放置
- 冷藏温度：0-4°C
- 冷冻温度：-18°C 以下
- 剩菜存放：2 小时内放冰箱，不超过 2 天

### 食物相克辟谣
- 螃蟹 + 柿子 ≠ 中毒（但鞣酸可能引起胃不适）
- 菠菜 + 豆腐 ≠ 结石（草酸和钙结合排出体外）
- 维生素 C + 虾 ≠ 砒霜（剂量远达不到中毒）

### 食物中毒症状
- 恶心、呕吐、腹痛、腹泻
- 发烧、头痛
- **处理**：立即停止食用，大量饮水，严重时就医

## 常见误区

| 误区 | 真相 |
|------|------|
| 土鸡蛋比洋鸡蛋更有营养 | 营养成分差异很小 |
| 喝骨头汤补钙 | 钙含量很低，不如喝牛奶 |
| 多吃水果减肥 | 水果含糖量高，需适量 |
| 隔夜水致癌 | 亚硝酸盐含量远低于安全线 |
| 味精有害健康 | 适量使用安全 |
| 红肉完全不能吃 | 适量摄入有益，过量有害 |
