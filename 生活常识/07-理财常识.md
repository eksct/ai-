# 理财常识

## 理财基础

### 核心原则
1. **收入 - 储蓄 = 支出**（先存后花，不是先花后存）
2. **紧急备用金**：3-6 个月的生活费，放在随时可取的地方
3. **不把鸡蛋放在一个篮子里**（分散投资）
4. **不懂不投**（不碰自己不了解的产品）
5. **复利效应**：越早开始储蓄投资，效果越明显

### 理财金字塔

```
        ▲
       ╱╲        高风险投资
      ╱  ╲       （股票、期货、加密货币）
     ╱    ╲
    ╱──────╲     成长型投资
   ╱        ╲    （基金、指数 ETF、股票）
  ╱──────────╲
 ╱            ╲  稳健型投资
╱──────────────╲ （债券、银行理财、储蓄保险）
▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔
  现金管理 + 保障
（活期存款、货币基金、保险）
```

## 预算与记账

### 50/30/20 法则
| 比例 | 用途 | 包含 |
|------|------|------|
| 50% | 必要支出 | 房租/房贷、吃饭、交通、水电 |
| 30% | 想要支出 | 娱乐、购物、旅行、外食 |
| 20% | 储蓄投资 | 存款、基金、保险、还债 |

### 记账方法
1. **记录每一笔支出**（APP 或手写）
2. **分类**：固定支出 vs 可变支出
3. **复盘**：月底看钱花在哪了
4. **优化**：减少不必要开支
5. **预算**：下月按预算执行

### 省钱误区
- ❌ 为了凑单免运费多买不需要的东西
- ❌ 因为打折而买（需要才买，不是便宜才买）
- ❌ 办了健身卡/视频会员但不用
- ❌ 每天一杯奶茶/咖啡（一年可超过 5000 元）

## 储蓄与存款

### 存款类型对比

| 类型 | 利率 | 灵活性 | 起存 |
|------|------|--------|------|
| 活期存款 | 0.2-0.35% | ⭐⭐⭐⭐⭐ | 1 元 |
| 定期存款（3 个月） | 1.1-1.3% | ⭐⭐ | 50 元 |
| 定期存款（1 年） | 1.5-1.7% | ⭐ | 50 元 |
| 定期存款（3 年） | 2.2-2.6% | ⭐ | 50 元 |
| 通知存款 | 0.8-1.0% | ⭐⭐⭐ | 5 万元 |
| 大额存单 | 高于定存 | ⭐ | 20 万元 |

### 推荐策略
- **备用金**：放货币基金（余额宝、零钱通），随取随用
- **短期目标**（1-2 年）：定期存款
- **中期目标**（3-5 年）：债券基金 + 纯债基金
- **长期目标**（5 年以上）：指数基金定投

## 投资工具

### 风险等级

```
低风险          →          高风险
存款 ↔ 货币基金 ↔ 债券 ↔ 混合基金 ↔ 股票基金 ↔ 股票 ↔ 期货/加密货币
```

### 基金种类

| 类型 | 风险 | 预期收益 | 说明 |
|------|------|----------|------|
| 货币基金 | 极低 | 1.5-2.5% | 余额宝、零钱通 |
| 短债基金 | 低 | 2-4% | 短期闲钱 |
| 纯债基金 | 中低 | 3-5% | 稳健理财 |
| 混合基金 | 中 | 5-10% | 股债搭配 |
| 指数基金 | 中高 | 8-15%（长期） | 定投首选 |
| 股票基金 | 高 | ±20%+ | 波动大 |

### 指数基金定投

**为什么定投？**
- 摊低成本（市场跌时买更多份额）
- 不用择时（克服人性弱点）
- 强制储蓄

**推荐标的：**
- 沪深 300（大盘蓝筹）
- 中证 500（中小盘）
- 科创 50（科技成长）
- 标普 500（美股）
- 纳斯达克 100（科技股）

**定投策略：**
- 每月固定日期、固定金额
- 坚持 3 年以上
- 市场低点多投，高点少投（或止盈）

### 股票投资原则（给普通投资者）
1. **闲钱投资**：3-5 年内用不上的钱
2. **分散持仓**：5-10 只不同行业股票
3. **长期持有**：不追涨杀跌
4. **不借钱炒股**
5. **不 all in 一只**

## 保险

### 购买顺序

```
社保（医保 + 养老）→ 百万医疗险 → 重疾险 → 意外险 → 寿险 → 理财险
```

### 各险种说明
| 险种 | 保什么 | 谁需要 | 保额建议 |
|------|--------|--------|----------|
| 百万医疗 | 大病住院报销 | 所有人都需要 | 200-400 万 |
| 重疾险 | 确诊即赔一笔钱 | 家庭支柱 | 年收入 3-5 倍 |
| 意外险 | 意外伤残/身故 | 经常出行的人 | 50-100 万 |
| 定期寿险 | 身故赔钱 | 有房贷/有孩子的人 | 房贷 + 5 年生活费 |
| 年金险 | 养老补充 | 有闲钱想规划养老 | 视情况 |

### 保险避坑
- ✅ 先保障后理财（别买"有病治病，没病返本"的返还型）
- ✅ 重疾险买够保额比买多家更重要
- ✅ 医疗险关注续保条件（选保证续保 20 年的）
- ❌ 不买捆绑销售（一个保单啥都保的往往都不划算）

## 房贷与信用卡

### 房贷
| 方式 | 说明 | 适合人群 |
|------|------|----------|
| 等额本息 | 每月还款金额相同，前期利息多 | 收入稳定 |
| 等额本金 | 每月还本金相同，总利息少 | 前期还款能力强 |

- 首套房贷利率：LPR ± 基点（目前 LPR 约 3.95%）
- 提前还款：看是否收违约金、是否在利率下降周期

### 信用卡
- **每月按时还款**，不逾期（影响征信）
- **最低还款额**利息高（日息万分之五，年化 18%+）
- **分期付款**实际利率约名义利率的 2 倍
- **不套现、不以卡养卡**
- 一年查 1-2 次征信报告

## 避坑指南

### 常见骗局
| 骗局 | 特征 | 防骗 |
|------|------|------|
| 杀猪盘 | 网恋 + 带投资 | 不转账 |
| 刷单兼职 | 先垫付后返利 | 刷单违法 |
| 高息理财 | 日收益 1%+ | 超过 6% 要警惕 |
| 虚拟货币炒币 | 拉群喊单 | 不参与 |
| 冒充客服退款 | 索要验证码 | 官方渠道核实 |
| 注销校园贷 | 说你征信有问题 | 征信找官方 |

### 投资骗局特征
- 承诺"保本高收益"（保本和高收益不可兼得）
- "名额有限、限量发售"制造紧迫感
- 发展下线有提成（传销特征）
- 公司信息查不到
- 把钱打入个人账户

## 理财建议

### 不同人生阶段的重点
| 阶段 | 重点 |
|------|------|
| 20-30 岁 | 存钱、建立应急基金、买保险、开始定投 |
| 30-40 岁 | 增加投资比例、考虑买房、教育金规划 |
| 40-50 岁 | 平衡投资、养老规划、控制风险 |
| 50 岁以上 | 降低风险、保证现金流、遗产规划 |

### 一句话原则
> **储蓄 = 收入 - 欲望，投资 = 时间 + 耐心，风险 = 未知 + 贪婪。**
