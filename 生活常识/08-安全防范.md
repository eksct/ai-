# 安全防范常识

## 防诈骗

### 十大常见诈骗类型

| 类型 | 手法 | 防骗要点 |
|------|------|----------|
| **刷单返利** | 先小额返利，后大额吞掉 | 凡是刷单都是诈骗 |
| **冒充客服** | 说退款/赔偿，索要验证码 | 主动联系官方核实 |
| **杀猪盘** | 网恋交友 + 诱导投资/赌博 | 不转账！ |
| **冒充公检法** | 说你涉嫌犯罪，要求转账到"安全账户" | 公检法不会电话办案 |
| **贷款诈骗** | 先交手续费/保证金才能放款 | 正规贷款不放款前收费 |
| **投资理财** | 高收益、稳赚不赔、老师带单 | 超过 6% 收益要警惕 |
| **冒充熟人** | 盗用微信/QQ 借钱 | 电话确认 |
| **中奖诈骗** | 你中奖了，先交税/手续费 | 先交钱的奖都是假的 |
| **注销校园贷** | 说影响征信，要求转账 | 征信自己查 |
| **裸聊敲诈** | 诱导裸聊后录屏敲诈 | 不裸聊！ |

### 防骗铁律
1. **不轻信**：陌生电话、短信、链接
2. **不透露**：验证码、密码、身份证号
3. **不转账**：未经核实绝不转账
4. **下载国家反诈中心 APP**：开启来电预警
5. **96110** 来电一定要接（反诈专线）

## 网络安全

### 密码安全
- 不同平台用不同密码
- 密码长度 ≥12 位，包含大小写字母 + 数字 + 符号
- 使用密码管理器（Bitwarden/1Password）
- 开启双重认证（2FA）
- 不在公共电脑保存密码

### 公共 Wi-Fi
- 不连接无密码的公共 Wi-Fi
- 不通过公共 Wi-Fi 进行网银操作
- 使用 VPN 加密通信
- 用完关掉 Wi-Fi

### 个人信息保护
- 不随意填写问卷调查
- 不参加需要身份证号的"免费领"
- 旧手机恢复出厂设置前：退出账号、多次覆盖写入数据
- 快递单、外卖单涂抹个人信息
- 社交媒体晒照注意背景中的敏感信息

### 手机安全
- 不安装来源不明的 APP
- 定期检查 APP 权限（关闭不必要的）
- 不随意扫陌生二维码
- 不 root/越狱手机
- 关闭"免密支付"功能

## 居家安全

### 门窗安全
- 出门前检查门窗是否锁好
- 低楼层安装防盗窗/防盗网
- 猫眼加装挡板（防止反窥）
- 门上装安全链
- 长期外出时请邻居帮忙收快递/开灯

### 燃气安全
- 每月检查燃气软管是否老化
- 燃气泄漏：**不开关电器、不点火**，开窗通风
- 燃气热水器不要装在浴室
- 定期请专业人员检查
- 安装燃气报警器

### 用电安全
- 不超负荷用电
- 充电线老化及时更换
- 手机充电时不要放在床上/枕头下
- 电热毯不整夜通电
- 电动车电池不在室内充电

### 火灾逃生
1. **保持冷静**，判断火势
2. **湿毛巾捂住口鼻**
3. **弯腰低姿**沿墙走（烟往上飘）
4. **不乘电梯**，走楼梯
5. **身上着火**：停、躺、滚（Stop, Drop, Roll）
6. **门烫不打开**：堵塞门缝，在窗口等待救援
7. **不跳楼**（3 层以上几乎没有生还希望）

## 出行安全

### 夜间出行
- 走大路、亮路
- 不戴耳机走路（降低警觉性）
- 把包背在里侧
- 打车确认车牌和车型再上车
- 分享行程给亲友

### 网约车安全
- 上车前核对车牌、车型、司机信息
- 坐后排
- 分享行程
- 不透露过多个人信息
- 发现异常要求下车或报警

### 公共场所
- 贵重物品随身携带，不放在椅背/桌上
- 背包背在前面或斜挎
- 手机拿在手里放在口袋
- 警惕拥挤场合（扒手）
- 随时观察周围环境

## 自然灾害应对

### 地震
- **室内**：躲在坚固桌子下/墙角，护住头部
- **室外**：远离建筑物、电线杆、广告牌
- **开车**：停到空旷处，留在车内
- **不要**：跳楼、乘电梯、跑到狭窄巷道
- 震后：检查燃气、水电，准备余震

### 台风/暴风雨
- 提前储备食物、水、手电筒、充电宝
- 加固门窗
- 不在广告牌、临时建筑下停留
- 避免外出
- 远离河岸、海岸

### 雷电
- 不在空旷地、山顶、水边停留
- 不站在大树下避雨
- 关掉电器、拔掉插头
- 不使用固定电话和淋浴
