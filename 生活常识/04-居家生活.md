# 居家生活常识

## 家居清洁

### 常用清洁剂及用途

| 清洁剂 | 用途 | 注意事项 |
|--------|------|----------|
| 84 消毒液 | 地板、马桶消毒 | 不能与洁厕灵混用（产生氯气） |
| 酒精（75%） | 表面消毒、电子产品 | 远离明火 |
| 白醋 | 水垢、玻璃 | 不能用于大理石表面 |
| 小苏打 | 去油污、除异味 | 可配合白醋使用 |
| 洗洁精 | 餐具、油污 | 充分冲洗 |
| 柠檬酸 | 除水垢（水壶、花洒） | 食品级安全 |

### 清洁频率建议
- **每日**：扫地、拖地、洗碗、整理床铺
- **每周**：换洗床单被套、擦窗台、清垃圾桶
- **每月**：清洗油烟机滤网、冰箱除霜、洗衣机清洗
- **每季**：洗窗帘、清空调滤网、清排水管
- **每半年**：清洗地毯、沙发深度清洁

### 去污小技巧
- **油污**：小苏打 + 温水浸泡
- **水垢**：白醋浸泡 30 分钟
- **茶渍/咖啡渍**：食盐搓洗
- **烧焦锅底**：小苏打 + 水煮沸
- **异味**：咖啡渣/小苏打放在冰箱/鞋柜

## 家电使用与保养

### 冰箱
- 冷藏室温度设为 2-4°C
- 冷冻室设为 -18°C
- 不要塞太满（影响冷气循环）
- 热食放凉后再放入
- 每月清理一次过期食物

### 空调
- 夏季制冷设为 26-27°C（最省电）
- 冬季制热设为 20-22°C
- 每月清洗滤网
- 定期请专业人员深度清洗
- 使用睡眠模式更省电

### 洗衣机
- 滚筒洗衣机每月自洁一次（倒入专用清洁剂/白醋）
- 洗完衣物打开门晾干
- 洗衣液不要过量（残留在密封圈）
- 定期清理排水过滤器

### 微波炉
- 不能用金属容器
- 加热液体时放木棍防止暴沸
- 鸡蛋不能带壳加热
- 加热后用湿布擦拭内部

## 衣物护理

### 洗衣标签符号

| 符号 | 含义 |
|------|------|
| 🏊 水盆 | 水洗温度 |
| △ | 可漂白 |
| □ 带圆圈 | 烘干 |
| ◎ | 干洗 |
| 🔥 | 熨烫温度 |

### 去渍方法
| 污渍 | 去渍方法 |
|------|----------|
| 油渍 | 洗洁精涂抹，静置 10 分钟后洗 |
| 血渍 | 冷水浸泡（热水会固定蛋白质） |
| 酱油 | 白醋搓洗 |
| 红酒 | 食盐覆盖吸收 |
| 果汁 | 热水 + 洗洁精 |
| 墨迹 | 酒精或牛奶浸泡 |
| 口红 | 酒精擦拭 |

### 衣物收纳
- **换季衣物**：洗净晒干，真空压缩袋保存
- **毛衣**：折叠存放，不要挂（变形）
- **西装**：挂装，用防尘罩
- **内衣袜子**：分隔收纳
- **防蛀**：樟脑丸/薰衣草香包（不要直接接触衣物）

## 食品储存

| 食物 | 储存方式 | 保质期 |
|------|----------|--------|
| 鸡蛋 | 冷藏，不水洗 | 30-45 天 |
| 牛奶 | 冷藏 2-4°C | 开封后 3-5 天 |
| 蔬菜 | 保鲜袋冷藏（绿叶菜根部包裹湿纸巾） | 3-7 天 |
| 水果 | 香蕉/苹果释放乙烯，分开存放 | 视种类定 |
| 肉类 | 冷冻 -18°C | 3-6 个月 |
| 调料 | 避光阴凉 | 开封后 6-12 个月 |
| 大米 | 密封避光，放花椒/大蒜防虫 | 6-12 个月 |

### 食物不宜放冰箱
- 土豆（低温让淀粉变甜）
- 洋葱（潮湿易发芽）
- 西红柿（破坏口感）
- 香蕉（变黑）
- 蜂蜜（结晶）
- 巧克力（起白霜）

## 省钱技巧

### 水电燃气
- 空调配风扇使用，减少空调负担
- 冰箱装 80% 满最省电
- 使用 LED 灯泡（比白炽灯省电 80%）
- 电器不使用时拔掉插头（待机仍耗电 5-10%）
- 集中洗涤：洗衣机尽量满负载运行
- 淋浴代替盆浴

### 购物技巧
- 列购物清单，避免冲动消费
- 关注消耗品价格，大促时囤货
- 不买"因为便宜"的东西，买"因为需要"的东西
- 必需品趁活动买，非必需品等需要买
- 比较单位价格（元/克、元/毫升）

## 安全常识

### 燃气安全
- 使用燃气时开窗通风
- 人不离灶，用完后关闭阀门
- 闻到家中有煤气味：**不开灯、不点火、不开电器**，开窗通风，到室外拨打燃气公司电话

### 用电安全
- 不要多个大功率电器共用一个插座
- 电热毯不要折叠使用
- 手机充电器不用时拔掉
- 手湿不碰电器开关
- 定期检查电线是否老化

### 防盗
- 出门反锁门
- 不要在家门口放备用钥匙（地毯下、花盆底）
- 快递单上的个人信息撕碎再扔
- 长时间离家可安装智能灯控模拟有人

### 防火
- 家中常备灭火器（干粉灭火器）
- 厨房油锅起火：盖锅盖，不要用水浇
- 每个房间有两条逃生路线
- 安装烟雾报警器
- 不私拉乱接电线
