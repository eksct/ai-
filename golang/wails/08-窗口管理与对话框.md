# 窗口管理与对话框

## 概述

桌面端和 Web 的最大区别：窗口位置、大小、对话框这些是你做图像管理 App 每天都要打交道的东西。

## 窗口配置

```go
wails.Run(&options.App{
    Title:  "ImageManager",
    Width:  1280,
    Height: 800,
    // 最小窗口尺寸（防止 UI 被挤变形）
    MinWidth:    900,
    MinHeight:   600,
    // 默认居中
    WindowStartState: options.Normal,       // Normal / Maximised / Minimised / Fullscreen
    // 是否可调大小
    Resizable: true,
    // 隐藏标题栏（自定义标题栏用）
    Frameless: false,
    // 置顶
    AlwaysOnTop: false,
})
```

### 前端控制窗口

```js
import { WindowSetSize, WindowSetTitle, WindowFullscreen, WindowUnfullscreen } from "../wailsjs/runtime";

// 全屏查看图片
function viewFullscreen() {
    WindowFullscreen();
}

// 退出全屏
function exitFullscreen() {
    WindowUnfullscreen();
}

// 设定窗口大小
WindowSetSize(800, 600);

// 窗口居中
import { WindowCenter } from "../wailsjs/runtime";
WindowCenter();
```

### Go 端控制窗口

```go
import "github.com/wailsapp/wails/v2/pkg/runtime"

// 设置窗口大小
runtime.WindowSetSize(a.ctx, 1280, 800)

// 获取窗口信息
w, h := runtime.WindowGetSize(a.ctx)   // 窗口尺寸
x, y := runtime.WindowGetPosition(a.ctx) // 窗口位置

// 全屏切换
runtime.WindowFullscreen(a.ctx)
runtime.WindowUnfullscreen(a.ctx)

// 窗口置顶
runtime.WindowSetAlwaysOnTop(a.ctx, true)
```

**记住窗口位置是桌面应用的标配：**
```go
func (a *App) OnShutdown(ctx context.Context) {
    // 保存窗口位置，下次启动恢复
    x, y := runtime.WindowGetPosition(ctx)
    w, h := runtime.WindowGetSize(ctx)
    settings := map[string]int{
        "window_x": x, "window_y": y,
        "window_w": w, "window_h": h,
    }
    a.saveSettings(settings)
}

func (a *App) OnStartup(ctx context.Context) {
    settings := a.loadSettings()
    runtime.WindowSetSize(ctx, settings["window_w"], settings["window_h"])
    runtime.WindowSetPosition(ctx, settings["window_x"], settings["window_y"])
}
```

## 文件对话框（图像 App 最常用）

这是你处理图片时第一个要用的功能。

### 打开文件

```go
runtime.OpenFileDialog(a.ctx, runtime.OpenDialogOptions{
    Title: "选择图片",
    Filters: []runtime.FileFilter{
        {
            DisplayName: "图片文件",
            Pattern:     "*.jpg;*.jpeg;*.png;*.gif;*.webp;*.bmp;*.tiff",
        },
        {
            DisplayName: "所有文件",
            Pattern:     "*.*",
        },
    },
    DefaultDirectory: a.lastOpenDir,      // 记住上次打开的目录
    DefaultFilename:  "",
    ShowHiddenFiles:  false,
    CanCreateDirectories: false,
    // 多选
    AllowsMultipleSelection: false,
})
```

### 多选文件

```go
// 批量导入图片
runtime.OpenFileDialog(a.ctx, runtime.OpenDialogOptions{
    Title: "批量导入图片",
    Filters: []runtime.FileFilter{
        {DisplayName: "图片文件", Pattern: "*.jpg;*.jpeg;*.png"},
    },
    AllowsMultipleSelection: true,  // 多选
})
```

### 保存文件

```go
runtime.SaveFileDialog(a.ctx, runtime.SaveDialogOptions{
    Title: "导出为",
    DefaultDirectory: a.lastExportDir,
    DefaultFilename:  "未命名.jpg",
    Filters: []runtime.FileFilter{
        {DisplayName: "JPEG 图片", Pattern: "*.jpg"},
        {DisplayName: "PNG 图片",  Pattern: "*.png"},
    },
})
```

### 选择目录

```go
// 让用户选一个文件夹（导入图片目录、导出目录）
directory := runtime.OpenDirectoryDialog(a.ctx, runtime.OpenDialogOptions{
    Title:            "选择图片目录",
    DefaultDirectory: a.lastImportDir,
})
```

## 系统菜单

```go
import "github.com/wailsapp/wails/v2/pkg/menu"
import "github.com/wailsapp/wails/v2/pkg/menu/keys"

AppMenu := menu.NewMenu()
FileMenu := AppMenu.AddSubMenu("文件")
FileMenu.AddText("打开...", keys.CmdOrCtrl("o"), func(_ *menu.CallbackData) {
    // 打开文件
})
FileMenu.AddText("导入目录...", keys.CmdOrCtrl("i"), func(_ *menu.CallbackData) {
    // 导入目录
})
FileMenu.AddSeparator()
FileMenu.AddText("导出...", keys.CmdOrCtrl("e"), func(_ *menu.CallbackData) {
    // 导出
})
FileMenu.AddSeparator()
FileMenu.AddText("退出", keys.CmdOrCtrl("q"), func(_ *menu.CallbackData) {
    runtime.Quit(a.ctx)
})

EditMenu := AppMenu.AddSubMenu("编辑")
EditMenu.AddText("撤销", keys.CmdOrCtrl("z"), nil)

ViewMenu.AddText("缩略图", nil, func(_ *menu.CallbackData) {
    runtime.EventsEmit(a.ctx, "view:change", "thumbnail")

    runtime.EventsEmit(a.ctx, "view:change", "list")

Menu: AppMenu,
```

## 踩坑记录

### 1. FileDialog 返回空路径
**原因：** 用户点了取消。返回空字符串不代表报错，需要判断：
```go
path, err := runtime.OpenFileDialog(a.ctx, opts)
if path == "" {
    return  // 用户点了取消，不是错误
}
```

### 2. Mac 菜单不显示中文
**原因：** 某些版本 Wails 的菜单中文支持有问题。如果你发现 Mac 上菜单乱码，用英文菜单项，或者等 Wails 修复。

### 3. Windows 上 WebView2 未安装
**现象：** 用户打开应用没反应，或者报错找不到 WebView2

**解决：** 在 build 配置里指定 WebView2 的部署方式：
```json
// wails.json
{
  "webview": "edgechromium",
  "webview2": "embed"  // 或者 download，embed 会增大包体
}
```
建议在安装包中附带 WebView2 安装程序，或者让用户安装 Microsoft Edge。
