# 测试

## 概述

Go 的测试是语言级集成的——不需要第三方测试框架。`go test` 自带测试、基准测试、覆盖率分析。

## 基本测试

```go
// filename_test.go（必须是 _test.go 结尾）
package processor

import "testing"

func TestResizeImage(t *testing.T) {
    result, err := ResizeImage("test.jpg", 800, 600)
    if err != nil {
        t.Fatalf("ResizeImage failed: %v", err)
    }
    if result.Bounds().Dx() != 800 {
        t.Errorf("expected width 800, got %d", result.Bounds().Dx())
    }
}
```

```bash
# 运行测试
go test                          # 当前包
go test ./...                    # 所有包
go test -v                       # 详细输出
go test -run TestResize          # 运行匹配的测试
go test -count=1                 # 禁用缓存
```

## Table-Driven Test

这是 Go 的测试风格：

```go
func TestParseImagePath(t *testing.T) {
    tests := []struct {
        name    string
        path    string
        want    *ImageInfo
        wantErr bool
    }{
        {
            name: "标准路径",
            path: "/images/photo.jpg",
            want: &ImageInfo{Name: "photo", Ext: ".jpg"},
        },
        {
            name: "含空格路径",
            path: "/my photos/vacation 2024.png",
            want: &ImageInfo{Name: "vacation 2024", Ext: ".png"},
        },
        {
            name: "无扩展名",
            path: "/images/photo",
            wantErr: true,
        },
    }

    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            got, err := ParseImagePath(tt.path)
            if tt.wantErr {
                if err == nil {
                    t.Error("expected error, got nil")
                }
                return
            }
            if err != nil {
                t.Fatalf("unexpected error: %v", err)
            }
            if !reflect.DeepEqual(got, tt.want) {
                t.Errorf("got %+v, want %+v", got, tt.want)
            }
        })
    }
}
```

## 子测试

```go
func TestImage(t *testing.T) {
    // setup
    img := loadTestImage()

    t.Run("resize", func(t *testing.T) {
        result := Resize(img, 800, 600)
        if result == nil {
            t.Fatal("result is nil")
        }
    })

    t.Run("crop", func(t *testing.T) {
        result := Crop(img, 100, 100, 500, 500)
        // 测试裁剪逻辑
    })

```

## Helper 函数

```go
func TestProcess(t *testing.T) {
    img := createTestImage(t)  // 如果 createTestImage 失败，调 t.Helper()

    result := Process(img)
    // ...
}

func createTestImage(t *testing.T) *Image {
    t.Helper()  // 标记为 helper，报错时行号指向调用方
    img, err := NewImage(100, 100)
    if err != nil {
        t.Fatal(err)
    }
    return img
}
```

## 基准测试（Benchmark）

```go
func BenchmarkResize(b *testing.B) {
    img := loadTestImage()
    b.ResetTimer()  // 忽略加载时间

    for i := 0; i < b.N; i++ {
        Resize(img, 800, 600)
    }
}

func BenchmarkResizeParallel(b *testing.B) {
    img := loadTestImage()
    b.RunParallel(func(pb *testing.PB) {
        for pb.Next() {
            Resize(img, 800, 600)
        }
    })
}
```

```bash
go test -bench=.                     # 跑所有 benchmark
go test -bench=. -benchmem           # 带内存分配统计
go test -bench=. -cpu=4,8           # 不同 CPU 数
```

输出：
```
BenchmarkResize-8        1000   1234567 ns/op   524288 B/op   12 allocs/op
```

## 覆盖率

## 测试辅助工具

### httptest（HTTP 测试）

```go
func TestHandleUpload(t *testing.T) {
    handler := http.HandlerFunc(uploadHandler)

    // 创建测试请求
    body := &bytes.Buffer{}
    writer := multipart.NewWriter(body)
    part, _ := writer.CreateFormFile("image", "test.jpg")
    part.Write([]byte("fake image data"))
    writer.Close()

    req := httptest.NewRequest("POST", "/upload", body)
    req.Header.Set("Content-Type", writer.FormDataContentType())
    rec := httptest.NewRecorder()

    handler.ServeHTTP(rec, req)

    if rec.Code != http.StatusOK {
        t.Errorf("expected 200, got %d", rec.Code)
    }
}
```

### iotest（IO 测试）

```go
import "testing/iotest"

func TestReadImage(t *testing.T) {
    data := []byte{/* ... */}

    // 模拟慢速读取
    reader := iotest.TimeoutReader(bytes.NewReader(data))
    _, err := ReadImage(reader)
    if err == nil {
        t.Error("expected timeout error")
    }
}
```

## Wails 项目的测试

```go
// 业务逻辑抽到独立的 service，不依赖 Wails runtime
type ImageService struct {
    // 不依赖 context
}

func (s *ImageService) Resize(path string, w, h int) error {
    // 纯业务逻辑，可以直接测试
}

// Wails App 只做桥接
type App struct {
    service *ImageService
}

func (a *App) ResizeImage(path string, w, h int) error {
    return a.service.Resize(path, w, h)
}
```

这样测试时只测 `ImageService`，不依赖 Wails 环境。
