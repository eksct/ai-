# 快捷键与高效操作

## 概述

如果你还在用鼠标点文件树找文件，说明你还没开始用 GoLand。IDE 的价值 80% 在快捷键上。

## 最常用的 10 个快捷键

| 快捷键 | 功能 | 一句话 |
|--------|------|--------|
| `Shift+Shift` | 全局搜索 | 搜文件、类、设置，什么都能搜 |
| `Ctrl+E` | 最近文件 | 在两个文件间快速切换 |
| `Ctrl+B` | 跳转定义 | 跳到函数/变量定义处 |
| `Ctrl+Alt+B` | 跳转实现 | 跳到接口的实现 |
| `Alt+F7` | 查找用法 | 看这个函数/变量在哪被用了 |
| `Ctrl+Shift+F` | 全局搜索文本 | 和普通搜索的区别：搜内容不是文件名 |
| `Ctrl+F12` | 文件结构 | 看当前文件的所有函数、类型 |
| `Alt+Insert` | 生成代码 | Getter、构造、测试文件 |
| `Ctrl+Shift+Enter` | 补全当前语句 | 自动加分号、大括号 |
| `F2` | 跳转到下一个错误 | 快速定位编译错误 |

## 导航

```text
Ctrl+Shift+A       查找 Action（配快捷键时好用）
Ctrl+N             按类名查找（Go 里就是按类型名查找）
Ctrl+Shift+N       按文件名查找
Ctrl+H             类型层次结构（看 struct 实现了哪些接口）
Alt+↑/↓            上一个/下一个方法
Ctrl+[ / ]         代码块开始/结束位置
Ctrl+Shift+Backspace 上次编辑位置
Ctrl+Alt+←/→       前后导航（浏览历史）
```

## 编辑

```text
Ctrl+D             复制当前行
Ctrl+Y             删除当前行
Ctrl+Shift+↑/↓     上移/下移当前行
Ctrl+/             注释/取消注释
Ctrl+Shift+U       大小写转换
Ctrl+Alt+L         格式化代码（保存时自动格式化就不用管这个）
Ctrl+Shift+V       从历史粘贴板选择粘贴
Alt+Shift+Insert   列编辑模式（选多行同时编辑）
```

## 代码补全

```text
Ctrl+Space         基本补全
Ctrl+Shift+Space   智能补全（会过滤出类型匹配的候选项）
Ctrl+Shift+Enter   补全并换行
Tab                接受补全（比 Enter 快，手不离开主键区）
Shift+F6           重命名（重构 - 所有引用一起改）
```

**GoLand 的 Postfix Completion：**
```go
// 输入变量名后加 . 加后缀
user.var    → user := User{}
user.err    → user, err := fn()
err.nnil    → if err != nil { return }
err.return  → if err != nil { return err }
user.len    → len(user)
```

## Live Templates（代码模板）

GoLand 内置了很多 Go 模板，也会自动提示：

```go
// 输入以下快捷键会展开
main     → func main() { ... }
fn       → func() { ... }
for      → for i := 0; i < n; i++ { ... }
forr     → for _, v := range slice { ... }
err      → if err != nil { return }
errp     → if err != nil { panic(err) }
go       → go func() { ... }()
gorr     → go func() { ... }() 并处理 err
```

你也可以自己加：
```
Settings → Editor → Live Templates → Go → +
```

比如我常用的 Wails 模板：
```go
// 缩写：wailsstart
func (a *App) startup(ctx context.Context) {
    a.ctx = ctx
}
```

## 使用时没必要做的操作

| 操作 | 替代方案 |
|------|----------|
| 用鼠标点文件树找文件 | `Ctrl+Shift+N` 搜文件名 |
| 手敲 import | 写完代码 `Ctrl+Alt+O` 自动优化 |
| 手动对齐代码 | `Ctrl+Alt+L` 格式化 |
| 手动重命名变量 | `Shift+F6` 所有引用一起改 |
| 手写 getter/setter | `Alt+Insert` → `Getter/Setter` |
| 切到终端敲 go build | 直接在 IDE 点绿色箭头运行 |
| 翻代码找函数定义 | `Ctrl+B` 跳转 |
