# 基础语法

## 变量

### 声明方式

```go
// 方式一：var 声明（包内可用）
var name string = "Alice"
var age = 25          // 类型推断
var count int         // 零值初始化（0）

// 方式二：短变量声明（只能在函数内）
name := "Alice"       // 最常用
age := 25

// 多变量
var x, y int = 1, 2
a, b := "hello", true

// 注意：Go 中声明了未使用的变量会编译不通过
var unused int        // 编译错误
_ := someFunc()       // _ 忽略返回值（可以）
```

### 零值

Go 的变量声明后不赋值也有默认值——没有"未初始化"状态：

| 类型 | 零值 |
|------|------|
| `int`, `float64` | `0` |
| `string` | `""` |
| `bool` | `false` |
| `pointer`, `slice`, `map`, `chan`, `func`, `interface` | `nil` |

## 基本数据类型

### 整数

```go
int, int8, int16, int32 (rune), int64
uint, uint8 (byte), uint16, uint32, uint64
```

- `int` 在 64 位系统上是 64 位，32 位系统上是 32 位
- `byte` = `uint8`（处理二进制数据用）
- `rune` = `int32`（Unicode 码点）

### 浮点数

```go
float32, float64
```

**重要：** 不要用 `==` 直接比较浮点数，精度问题：
```go
0.1 + 0.2 == 0.3  // false！
```

### 布尔

```go
var ok bool = true
// 和 C 不同，Go 中 bool 不能和 int 互转
// if (1) { }      // 编译错误
```

### 字符串

```go
s := "hello"
s[0]               // 104（byte，不是字符）
len(s)             // 5
s + " world"       // 拼接

// 反引号原始字符串
s := `line1
line2`             // 包含换行

// rune 遍历（处理中文）
for i, r := range "你好" {
    fmt.Printf("%d: %c\n", i, r)  // 0: 你, 3: 好
}
```

### 类型转换

Go 没有隐式类型转换，必须显式：

```go
var i int = 42
var f float64 = float64(i)     // 显式转换
var u uint = uint(f)

// 字符串和数字互转
strconv.Itoa(42)               // "42"
strconv.Atoi("42")             // 42, nil
strconv.ParseFloat("3.14", 64) // 3.14, nil
```

## 常量

```go
const Pi = 3.14159
const (
    StatusOK  = 200
    StatusNotFound = 404
)

// iota 枚举
const (
    Monday = iota + 1  // 1
    Tuesday            // 2
    Wednesday          // 3
)
```

## 运算符

```go
+ - * / %             // 算术
== != < > <= >=       // 比较
&& || !               // 逻辑
& | ^ << >>           // 位运算
&^                     // 位清除（AND NOT）
```

**注意：** `++` 和 `--` 是语句不是表达式：
```go
i++       // 正确
j = i++   // 编译错误！
```

## 控制流

### if

```go
if x > 0 {
    fmt.Println("正数")
} else if x < 0 {
    fmt.Println("负数")
} else {
    fmt.Println("零")
}

// if 中可以写一个初始化语句（常用）
if err := doSomething(); err != nil {
    return err
}
// err 的作用域只在 if 块内
```

### for（Go 只有 for，没有 while）

```go
// 标准 for
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// 当 while 用
for x < 100 {
    x *= 2
}

// 无限循环
for {
    // 直到 break
}

// range 遍历
for i, v := range []int{1, 2, 3} {
    fmt.Println(i, v)
}

for k, v := range map[string]int{"a": 1} {
    fmt.Println(k, v)
}

// 只遍历 key 或 value
for k := range m { }      // 只取 key
for _, v := range arr { } // 只取 value
```

### switch

```go
// 不需要 break，不会 fallthrough
switch os := runtime.GOOS; os {
case "darwin":
    fmt.Println("macOS")
case "linux":
    fmt.Println("Linux")
default:
    fmt.Println("其他")
}

// switch 可以没有条件（代替 if-else 链）
score := 85
switch {
case score >= 90:
    grade = "A"
case score >= 80:
    grade = "B"
default:
    grade = "C"
}
```

### defer

```go
// defer 在函数返回时执行（后进先出）
func readFile(path string) error {
    f, err := os.Open(path)
    if err != nil {
        return err
    }
    defer f.Close()  // 函数返回时执行

    // 处理文件...
    return nil
}
```

**defer 常见用途：**
```go
// 1. 关闭资源
defer f.Close()
defer resp.Body.Close()

// 2. 解锁
mu.Lock()
defer mu.Unlock()

// 3. 记录函数耗时
defer func() {
    log.Printf("耗时: %v", time.Since(start))
}()
```

**注意：** defer 的参数在声明时求值，不是执行时：
```go
x := 1
defer fmt.Println(x)  // 输出 1
x = 2
// 函数返回时打印 1，不是 2
```

### 标签和跳转

```go
// break 到外层
outer:
for i := 0; i < 10; i++ {
    for j := 0; j < 10; j++ {
        if j == 5 {
            break outer  // 跳出两层循环
        }
    }
}

// goto（极少用，但有些场景比 break 清晰）
if err != nil {
    goto cleanup
}
cleanup:
    closeResources()
```
