# IO 与网络编程

## IO.Reader / IO.Writer

Go 的 IO 围绕两个核心接口：

```go
type Reader interface {
    Read(p []byte) (n int, err error)
}

type Writer interface {
    Write(p []byte) (n int, err error)
}
```

**强大之处：** 所有涉及读写操作的类型都实现这两个接口（文件、网络、压缩、加密、内存缓冲区）。

```go
// 从文件读
f, _ := os.Open("photo.jpg")
data, _ := io.ReadAll(f)     // 从 Reader 读全部

// 写文件
f, _ := os.Create("output.jpg")
io.Copy(f, reader)           // 从 Reader 复制到 Writer

// 限制读取大小
r := io.LimitReader(file, 1024)  // 只读 1024 字节

// 组合 Reader/Writer
tr := io.TeeReader(reader, writer)  // 读的同时写入另一个 Writer
```

## 文件 IO

```go
// 读文件
data := os.ReadFile("path")           // 一次性读（小文件）
f := os.Open("path")                   // 流式读（大文件）
defer f.Close()
reader := bufio.NewReader(f)           // 带缓冲

// 写文件
os.WriteFile("path", data, 0644)       // 一次性写
f := os.Create("path")                  // 流式写
f.WriteString("hello")
f.Write(data)

// 临时文件
tmp, _ := os.CreateTemp("", "prefix-*")
defer os.Remove(tmp.Name())

// 遍历目录
entries, _ := os.ReadDir("dir/")
for _, entry := range entries {
    info, _ := entry.Info()
    fmt.Println(entry.Name(), info.Size())
}
```

## 网络编程

### TCP

```go
// 服务端
listener, _ := net.Listen("tcp", ":8080")
for {
    conn, _ := listener.Accept()
    go handleConn(conn)  // 每个连接一个 goroutine
}

// 客户端
conn, _ := net.Dial("tcp", "localhost:8080")
conn.Write([]byte("hello"))
```

```go

resp, _ := http.Get("https://api.example.com/data")
```

## 文件监控（fsnotify）

你的图像管理 App 需要监听目录变化：

```go
import "github.com/fsnotify/fsnotify"

watcher, _ := fsnotify.NewWatcher()
defer watcher.Close()

go func() {
    for {
        select {
        case event, ok := <-watcher.Events:
            if !ok { return }
            switch {
            case event.Has(fsnotify.Create):
                log.Printf("新文件: %s", event.Name)
            case event.Has(fsnotify.Remove):
                log.Printf("文件删除: %s", event.Name)
            case event.Has(fsnotify.Rename):
                log.Printf("文件重命名: %s", event.Name)
            }
        case err, ok := <-watcher.Errors:
            if !ok { return }
            log.Printf("监听错误: %v", err)
        }
    }
}()

watcher.Add("/path/to/watch")
```

**注意防抖：** 复制一个大文件会触发多次 Create 事件，需要用 `time.AfterFunc` 做防抖。

## 图片处理（标准库）

```go
import (
    "image"
    "image/jpeg"
    "image/png"
    "image/gif"
    _ "image/gif"   // 注册解码器
)

// 解码
f, _ := os.Open("photo.jpg")
defer f.Close()

// 只读头部（不加载像素）
cfg, _ := image.DecodeConfig(f)
fmt.Printf("尺寸: %dx%d, 格式: %s", cfg.Width, cfg.Height, cfg.Format)

// 解码全部
f.Seek(0, 0)  // 回到文件头
img, format, _ := image.Decode(f)

// 编码（写入文件）
out, _ := os.Create("output.jpg")
defer out.Close()
jpeg.Encode(out, img, &jpeg.Options{Quality: 85})
```
