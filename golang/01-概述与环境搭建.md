# Go 语言概述与环境搭建

## Go 是什么

Go（Golang）是 Google 2009 年推出的编译型静态语言。设计者是 Ken Thompson（Unix/C 作者）、Rob Pike（UTF-8 作者）和 Robert Griesemer。

**核心特点：**
- 编译型，直接生成二进制，无虚拟机
- 静态类型 + 类型推断
- 原生并发（goroutine + channel）
- 极简语法（只有 25 个关键字）
- 内置工具链（fmt、test、pprof、race）

## 为什么选择 Go

| 场景 | 为什么用 Go |
|------|-------------|
| 后端 API | 编译快、部署简单（单二进制） |
| 桌面应用（Wails） | 包体小、跨平台、WebView 绑定 |
| CLI 工具 | 交叉编译、静态链接 |
| 云原生中间件 | Docker、Kubernetes、etcd 都是用 Go 写的 |
| 图像处理 | 标准库自带 `image` 包，底层性能好 |

**和 Java 的对比：**
- Java 是"你写一次，到处运行"（JVM）
- Go 是"你编译一次，到处运行"（静态链接，单二进制）

## 安装

```bash
# 下载
https://go.dev/dl/

# 验证
go version

# 环境变量（通常 Go 1.16+ 不用配 GOPATH 也能用）
go env GOPATH   # 默认 ~/go
go env GOROOT   # Go 安装目录
```

### 国内代理（必须配）

```bash
go env -w GOPROXY=https://goproxy.cn,https://goproxy.io,direct
```

不配的话，`go mod download` 慢到怀疑人生。

## Hello World

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, 世界")
}
```

```bash
go run main.go         # 直接运行，不产生二进制
go build main.go       # 编译成二进制
./main                 # 运行
```

## GOPATH vs Modules

### 旧模式（GOPATH，Go 1.11 之前）

所有项目都在 `$GOPATH/src` 下，依赖管理靠 `go get`，没有版本——"依赖地狱"。

### 新模式（Modules，Go 1.16+ 默认）

```bash
go mod init github.com/you/project   # 初始化 module
go mod tidy                          # 整理依赖
go get github.com/xxx/package@v1.0  # 安装指定版本
```

每个项目有自己的 `go.mod`，依赖版本锁定，可复现构建。

## 编译

```bash
# 基本编译
go build -o myapp main.go

# 交叉编译（关键：Wails 打包跨平台）
GOOS=linux GOARCH=amd64 go build main.go
GOOS=windows GOARCH=amd64 go build main.go
GOOS=darwin GOARCH=amd64 go build main.go

# 编译优化
go build -ldflags="-s -w" main.go     # 去掉符号表和调试信息（减小体积）
go build -race main.go                # 带竞态检测（开发时用）

# 安装
go install main.go                     # 编译到 $GOPATH/bin
```

## Go 设计哲学

### 少就是多

- 没有类（用 struct + method 代替）
- 没有继承（用组合 + 接口代替）
- 没有泛型（Go 1.18 之前，1.18 终于有了）
- 没有异常（用 error 返回值代替）
- 没有构造函数
- 没有注解/装饰器（用接口组合代替）

### 显式优于隐式

```go
// Java: 可能抛异常，调用方可能忘记 catch
void foo() throws IOException { ... }

// Go: error 是返回值，调用方必须处理（或者显式忽略）
func foo() error { ... }
```

### 组合优于继承

```go
// Java 的继承
class Dog extends Animal {}

// Go 的组合
type Dog struct {
    Animal  // 嵌入 Animal 的方法和字段
}
```

go run      # 直接运行
