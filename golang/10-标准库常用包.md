# 标准库常用包

## 概述

Go 的标准库非常强大——很多第三方库的功能其实标准库已经提供了。每次想加依赖前，先翻翻标准库有没有。

## fmt（格式化 I/O）

```go
fmt.Print("hello")            // 无换行
fmt.Println("hello")          // 带换行
fmt.Printf("hello %s", "world") // 格式化

// Sprintf：返回字符串
s := fmt.Sprintf("name=%s, age=%d", "Alice", 25)

// 错误格式化
err := fmt.Errorf("user %d not found", id)

// % 动词
%v     // 值的默认格式
%+v    // 结构体带字段名
%#v    // Go 语法表示
%T     // 类型
%d     // 十进制整数
%s     // 字符串
%f     // 浮点数
%.2f   // 两位小数
%t     // 布尔
%p     // 指针
```

## io / os（文件操作）

```go
// 读文件
data, err := os.ReadFile("file.txt")      // 小文件
f, err := os.Open("file.txt")             // 大文件
defer f.Close()
scanner := bufio.NewScanner(f)             // 按行读
for scanner.Scan() {
    fmt.Println(scanner.Text())
}

// 写文件
os.WriteFile("file.txt", []byte("hello"), 0644)
f, _ := os.Create("file.txt")
defer f.Close()
f.WriteString("hello")

// 文件信息
os.Stat("file.txt")
os.IsNotExist(err)
os.MkdirAll("path/to/dir", 0755)   // 递归创建目录
os.Remove("file.txt")
os.RemoveAll("dir/")                // 递归删除

// 路径
filepath.Join("dir", "sub", "file.txt") // 跨平台路径拼接
filepath.Base("/a/b/c.txt")            // "c.txt"
filepath.Dir("/a/b/c.txt")             // "/a/b"
filepath.Ext("c.txt")                  // ".txt"
filepath.Glob("images/*.jpg")          // 通配符匹配
```

## time（时间）

```go
// 当前时间
now := time.Now()
now.Unix()                     // 时间戳（秒）
now.UnixMilli()                 // 毫秒
now.UnixNano()                  // 纳秒

// 构造时间
t := time.Date(2024, 1, 1, 0, 0, 0, 0, time.UTC)

// 格式化（layout 必须是这个时间点，不能改）
t.Format("2006-01-02 15:04:05")   // "2024-01-01 00:00:00"
t.Format("2006-01-02")              // "2024-01-01"

// 解析
t, _ := time.Parse("2006-01-02", "2024-01-01")

// 时间计算
t.Add(2 * time.Hour)
t.AddDate(1, 0, 0)              // 加一年
t.Before(deadline)
t.After(start)

// 定时器
timer := time.NewTimer(1 * time.Second)
<-timer.C                        // 1 秒后触发

ticker := time.NewTicker(1 * time.Second)
for range ticker.C {
    // 每秒执行一次
}

// 超时
select {
case <-ch:
    // 收到数据
case <-time.After(5 * time.Second):
    // 超时
}

// 耗时计算
start := time.Now()
doSomething()
elapsed := time.Since(start)
fmt.Println(elapsed)
```

## encoding/json（JSON）

```go
type Person struct {
    Name string `json:"name"`
    Age  int    `json:"age"`
}

// 序列化
p := Person{Name: "Alice", Age: 25}
data, _ := json.Marshal(p)             // []byte
data, _ := json.MarshalIndent(p, "", "  ")  // 格式化

// 反序列化
var p Person
json.Unmarshal([]byte(`{"name":"Alice"}`), &p)

// 流式读写（大 JSON）
decoder := json.NewDecoder(file)
for decoder.More() {
    var item Item
    decoder.Decode(&item)
}

// 任意 JSON
var data interface{}
json.Unmarshal([]byte(jsonStr), &data)
// data 是 map[string]interface{}
```

## strings / strconv

```go
// strings
strings.Contains(s, "sub")
strings.HasPrefix(s, "http")
strings.HasSuffix(s, ".jpg")
strings.Index(s, "@")           // 位置
strings.Split("a,b,c", ",")     // ["a","b","c"]
strings.Join([]string{"a","b"}, ",") // "a,b"
strings.TrimSpace(s)
strings.ToLower(s)
strings.ReplaceAll(s, "old", "new")
strings.Repeat("x", 3)           // "xxx"

// strconv
strconv.Itoa(42)                  // "42"
strconv.Atoi("42")                // 42, nil
strconv.ParseFloat("3.14", 64)    // 3.14, nil
strconv.ParseBool("true")         // true, nil
```

## sort

```go
s := []int{3, 1, 4, 1, 5}
sort.Ints(s)                    // 升序
sort.Sort(sort.Reverse(sort.IntSlice(s))) // 降序

// 自定义排序
sort.Slice(users, func(i, j int) bool {
    return users[i].Age > users[j].Age
})

// 查找（已排序的切片）
index := sort.SearchInts(s, 4)
```

## log

```go
log.Println("info")
log.Printf("user %d logged in", id)
log.Fatalf("fatal: %v", err)    // 打印后 os.Exit(1)

// 配置日志格式
log.SetFlags(log.Ldate | log.Ltime | log.Lshortfile)
log.SetOutput(os.Stdout)

// 结构化日志（Go 1.21+）
slog.Info("user login", "user_id", id, "ip", ip)
slog.Error("failed", "error", err)
```

## flag（命令行参数）

```go

flag.Parse()

fmt.Println(*port, *debug)

// 运行：./app -port=9090 -debug
```

## net/http（HTTP）

```go
// 服务端
http.HandleFunc("/api/hello", func(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"message": "hello"})
})
log.Fatal(http.ListenAndServe(":8080", nil))

// 客户端
resp, _ := http.Get("https://api.example.com/data")
defer resp.Body.Close()
body, _ := io.ReadAll(resp.Body)
```
